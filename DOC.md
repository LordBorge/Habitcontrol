# HabitControl Project

This is the HabitControl project. Follow the instructions below to get the project running on your local machine.

## Prerequisites

Make sure you have the following installed on your system:

- [Node.js](https://nodejs.org/) (includes npm)
- [Expo CLI](https://docs.expo.dev/get-started/installation/)
- [Git](https://git-scm.com/)

## Setup Instructions

### 1. Clone the Repository

(Before you do step 1, make sure to configure the environment variables on your device.)  

```git clone https://github.com/lordborge/habitcontrol.git```

### 2. Install dependencies 


```npm install```

### 3. Run the project 


```npx expo start```

###### If you encounter  any error, try resetting Node cache with 

```npm start -- --reset-cache```
